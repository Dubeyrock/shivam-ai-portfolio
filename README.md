# Shivam AI Portfolio

Advanced Next.js portfolio for Shivam Dubey with:
- 3D-style developer theme
- Dark/light mode
- Project filtering
- Contact form
- Skills progress bars
- Social media section
- Timeline
- Admin dashboard for add/update/delete/upload
- Visitor analytics hooks

## Run locally

```bash
npm install
npm run dev
```
